# ✨ BE GLOW 쇼핑몰 홈페이지 리뉴얼 ✨
<br/>

## 🛠️ 프로젝트 정보
**BE GLOW**는 화장품 및 라이프스타일 제품을 전문으로 판매하는 쇼핑몰로,  
시각적 집중도와 사용자 중심 UI/UX를 목표로 리뉴얼 하였습니다.  
심플하지만 직관적인 레이아웃과 상품 중심의 콘텐츠 구성으로 사용자 경험을 극대화하였습니다.
### 개발 기간
- 2025.05.23 ~ 2025.05.30
